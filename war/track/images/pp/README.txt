Some of the icons/pushpins in this directory were made using the "Marker Maker"
tool from GMapLive.  At the time many of these icons/pushpins were created, the
"Marker Maker" tools was available at the following link:
  http://www.gmaplive.com/marker_maker.php
However, it appears that this link is no longer available.
This "Marker Maker" tool had maintained the following copyright:
    Copyright 2006 Dixon Information Systems, Canada. All Rights Reserved.

Some of the icons/pushpins were created using other image editing tools, and
some have been provided by the following contributors:
  - Jason Herring 
    (pin30_teal_h*.png, pin30_orange_h*.png, pin30_lime_h*.png, pin30_gray_h*.png, 
    pin30_darkblue_h*.png, pin30_blue_h*.png, etc.)
